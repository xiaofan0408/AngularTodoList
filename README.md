# Aangular学习大纲

## 开发环境

- [Node.js](https://nodejs.org/zh-cn/) >= 8.9.0
- [Visual Studio Code](https://code.visualstudio.com/) (编辑器)




- [TypeScript](https://www.tslang.cn)
```bash
npm install -g typescript
```
- [Anuglar CLI](https://cli.angular.io/)
```bash
npm install -g @angular/cli
```
